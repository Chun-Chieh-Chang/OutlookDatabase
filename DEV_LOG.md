# DEV_LOG: 系統進化與故障確效紀錄 (2026-05-09)

## 1. 故障事件: 編碼回歸屠殺 (Regression Error)
* **現象**: 全域 .md 檔案出現大量問號亂碼。
* **RCA**: PowerShell Get-Content 預設以 CP950 讀取 UTF-8 檔案，導致原始數據損毀後寫回。
* **CAPA**: 禁止使用系統層級全域轉碼指令；改用 Python codecs 進行二進位流處理。

## 2. 故障事件: 公式渲染失能
* **現象**: LaTeX 公式顯示為原始代碼。
* **RCA**: 前端解析器 (marked.js) 缺乏數學引擎支援。
* **CAPA**: 引入 KaTeX 渲染引擎，並將手冊公式簡化為純 Markdown 語法以提升兼容性。

## 3. 工業架構進化: 多維度映射升級
* **變更**: 廢除 4M1E 單一限制，升級為包含 HR, PQC, QMS, CAPA 的 21 維度工業語意分層。
* **確效**: 透過 wiki_index_gen.py 驗證索引生成正確，且具備自發現維度能力。
