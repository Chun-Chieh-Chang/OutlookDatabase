### 1. 📧 郵件資料管理
- **自動提取**: 從 Outlook 收件匣提取郵件

- **重複檢查**: 避免重複儲存相同郵件

### 2. 🤖 AI 智慧分析
- **郵件摘要**: 自動生成郵件要點
- **智慧分類**: 8種郵件類型自動識別
- **關鍵字提取**: 自動提取重要關鍵詞
- **情感分析**: 識別郵件情感傾向

### 3. 🌐 網頁操作介面
- **現代化設計**: 響應式網頁介面
- **即時搜尋**: 關鍵字搜尋郵件
- **統計分析**: 郵件統計和趨勢
- **AI 整合**: 一鍵 AI 分析功能

## 📱 使用方式

### 啟動系統
```bash
# 1. 啟動 Ollama 服務（如未自動啟動）
ollama serve

# 2. 啟動網頁介面
python web_app.py

# 3. 開啟瀏覽器
http://localhost:5000
```

### 網頁介面操作

#### 🏠 主頁面功能
1. **資料庫狀態**: 檢查郵件資料庫狀態
2. **AI 狀態**: 檢查 AI 服務可用性
3. **建立資料庫**: 從 Outlook 提取郵件
4. **設定 AI**: 一鍵安裝 AI 模型

#### 🔍 搜尋功能
- 輸入關鍵字搜尋郵件主旨和內容
- 支援中文搜尋
- 即時顯示搜尋結果

#### 🤖 AI 分析
- **分析最近郵件**: 一鍵分析最近郵件
- **檢查 AI 狀態**: 確認 AI 服務狀態
- **查看分析結果**: 詳細的 AI 分析報告

#### 📊 統計資訊
- **總郵件數**: 資料庫中郵件總數
- **主要寄件者**: 最活躍的寄件者排行
- **最近郵件**: 最新收到的郵件列表

### AI 分析詳情

#### 📝 郵件摘要
- 自動生成郵件要點摘要
- 識別郵件重要性
- 建議回覆優先級

#### 🏷️ 智慧分類
1. **工作相關**: 專案、會議、任務
2. **客戶溝通**: 客戶詢問、報價、支援
3. **財務相關**: 發票、付款、預算
4. **人事通知**: 請假、公告、培訓
5. **技術支援**: IT問題、系統維護
6. **行銷推廣**: 產品介紹、活動通知
7. **個人郵件**: 朋友、家人郵件
8. **其他**: 無法明確分類

#### 🔍 關鍵字提取
- 自動識別重要關鍵詞
- 支援中文關鍵字
- 最多提取 10 個關鍵詞

#### 😊 情感分析
- **情感類型**: 正面/中性/負面
- **情感強度**: 1-10 分評估
- **關鍵詞識別**: 主要情感詞彙

## 🛠️ 故障排除

### 常見問題解決

#### AI 服務問題
**問題**: AI 狀態顯示「未設定」
**解決**: 
```bash
python check_ollama.py
python direct_ai_setup.py
```

#### 模型載入問題
**問題**: AI 分析超時
**解決**:
1. 重啟 Ollama 服務
2. 檢查記憶體使用量
3. 等待模型完全載入

#### 網頁介面問題
**問題**: 網頁無法載入
**解決**:
1. 確認 Flask 已安裝
2. 檢查 5000 埠是否被佔用
3. 重新啟動 web_app.py

### 效能優化建議

1. **記憶體配置**: 建議 16GB+ RAM
2. **模型選擇**: 可考慮使用更小的模型
3. **批次處理**: 一次分析多封郵件
4. **內容限制**: 限制分析內文長度
