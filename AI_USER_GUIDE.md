# 🤖 AI 功能使用指南

## 📋 功能總覽

### 🧠 智慧搜尋
- **語意搜尋** - 使用 AI 理解搜尋意圖，找到語意相關的郵件
- **關鍵字搜尋** - 傳統關鍵字匹配郵件主旨和內容
- **摘要搜尋** - 按郵件內容摘要進行搜尋（開發中）
- **分類搜尋** - 按郵件類型搜尋（開發中）

### 🛠️ 實用 AI 工具
- **分析最近郵件** - AI 分析最新 10 封郵件，提供摘要、分類、關鍵字、情感分析
- **分析搜尋結果** - AI 深度分析當前搜尋結果
- **生成郵件摘要** - 手動輸入郵件內容，AI 生成摘要
- **提取重點資訊** - AI 提取郵件關鍵字和重要資訊
- **快速回覆建議** - AI 生成專業回覆建議
- **多語言翻譯** - AI 翻譯郵件到指定語言
- **郵件洞察分析** - AI 分析整體郵件趨勢和統計

## 🚀 操作步驟詳解

### 1️⃣ 啟動系統
```bash
# 1. 確保 Ollama 服務運行
# 檢查：http://localhost:11434

# 2. 啟動網頁應用
python web_app.py

# 3. 開啟瀏覽器
# 訪問：http://localhost:5000
```

### 2️⃣ 智慧搜尋操作

#### 🧠 語意搜尋
1. **選擇搜尋類型**：點擊「🧠 語意搜尋」按鈕
2. **輸入自然語言**：例如「關於會議的郵件」、「本月的報告」
3. **執行搜尋**：點擊搜尋按鈕
4. **查看結果**：
   - 🏷️ 相關性分數（6-10分）
   - 📊 按相關性排序
   - 🎯 精準匹配搜尋意圖

#### 🔍 關鍵字搜尋
1. **選擇搜尋類型**：點擊「🔍 關鍵字搜尋」按鈕
2. **輸入關鍵字**：例如「會議」、「報告」、「客戶」
3. **執行搜尋**：點擊搜尋按鈕
4. **查看結果**：傳統關鍵字匹配結果

### 3️⃣ AI 工具使用

#### 📊 分析最近郵件
1. **點擊工具**：「🧠 分析最近郵件」
2. **等待分析**：AI 自動分析最新 10 封郵件
3. **查看報告**：
   - 📝 郵件摘要
   - 🏷️ 智慧分類
   - 🔍 關鍵字提取
   - 😊 情感分析
   - 📈 統計報告

#### 🔍 分析搜尋結果
1. **先進行搜尋**：使用任一搜尋方式找到郵件
2. **點擊分析**：「🔍 分析搜尋結果」
3. **深度分析**：AI 針對搜尋結果提供專業分析
4. **查看洞察**：了解搜尋結果的模式和趨勢

#### 📝 生成郵件摘要
1. **點擊工具**：「📝 生成郵件摘要」
2. **輸入內容**：
   - 主旨（必填）
   - 內容（選填）
3. **獲取摘要**：AI 生成精簡摘要

#### 🔑 提取重點資訊
1. **點擊工具**：「🔑 提取重點資訊」
2. **輸入郵件**：主旨和內容
3. **獲取結果**：
   - 🏷️ 關鍵字列表
   - 📋 重要資訊摘要

#### 💬 快速回覆建議
1. **點擊工具**：「💬 快速回覆建議」
2. **輸入原郵件**：主旨和內容
3. **獲取建議**：AI 提供 2-3 種回覆選項

#### 🌍 多語言翻譯
1. **點擊工具**：「🌍 多語言翻譯」
2. **輸入資訊**：
   - 郵件主旨
   - 郵件內容
   - 目標語言（如：英文、日文、韓文）
3. **獲取翻譯**：AI 提供準確翻譯

#### 📈 郵件洞察分析
1. **點擊工具**：「📈 郵件洞察分析」
2. **等待分析**：AI 分析整體郵件數據
3. **查看洞察**：
   - 📊 通訊趨勢
   - 👥 主要寄件者
   - 📅 時間模式
   - 💡 改善建議

### 4️⃣ 進階功能

#### 📥 匯出搜尋結果
1. **進行搜尋**：使用任一搜尋方式
2. **點擊匯出**：「📥 匯出搜尋結果」
3. **確認匯出**：檢查匯出資訊對話框
4. **下載檔案**：自動下載 CSV 檔案

#### 🎯 方案選擇提取
1. **選擇方案**：
   - 🚀 超快速測試（10封）
   - ⚡ 日常使用（50封）
   - 🔄 完整更新（200封）
   - 📊 完整匯入（500封）
   - 🔧 自訂設定
2. **執行提取**：點擊「📊 提取郵件」
3. **查看進度**：即時顯示提取進度和結果

## 🔧 故障排除

### ❌ 常見問題

#### AI 服務不可用
**症狀**：顯示「AI 未設定」
**解決**：
1. 檢查 Ollama 服務：`curl http://localhost:11434/api/tags`
2. 重啟 Ollama 服務
3. 重新載入網頁

#### 語意搜尋無結果
**症狀**：語意搜尋返回空結果
**解決**：
1. 嘗試更詳細的搜尋描述
2. 檢查郵件資料庫是否有內容
3. 使用關鍵字搜尋作為備選

#### AI 工具回應錯誤
**症狀**：AI 工具返回錯誤訊息
**解決**：
1. 檢查網路連接
2. 確認 Ollama 服務正常
3. 重新啟動 web_app.py

#### 搜尋結果無法匯出
**症狀**：匯出按鈕無反應
**解決**：
1. 確認已進行搜尋
2. 檢查搜尋結果是否顯示
3. 重新進行搜尋後再嘗試匯出
