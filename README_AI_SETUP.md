# 本地 AI 模型整合指南

## 🎯 概述

本指南將幫助您在 Outlook 郵件資料庫工具中整合本地開源 AI 模型，實現無需 API Key 的智慧分析功能。

## 📋 系統需求

### 最低配置
- **作業系統**: Windows 10/11 (64位)
- **記憶體**: 8GB RAM
- **儲存空間**: 10GB 可用空間
- **網路**: 穩定的網際網路連接（用於下載模型）

### 推薦配置
- **記憶體**: 16GB+ RAM
- **GPU**: NVIDIA GPU (支援 CUDA)
- **儲存空間**: 20GB+ SSD

## 🚀 安裝步驟

### 第一步：安裝 Python 依賴
```bash
pip install -r requirements.txt
```

### 第二步：執行 AI 安裝腳本
```bash
python setup_ai.py
```

這個腳本會自動：
1. 檢查系統環境
2. 下載並安裝 Ollama
3. 下載 Llama 3 模型 (約 4.7GB)
4. 設定配置檔案
5. 測試模型運行

### 第三步：測試 AI 功能
```bash
python email_analyzer.py
```

### 第四步：啟動網頁介面
```bash
python web_app.py
```

## 📱 使用 AI 功能

### 網頁介面操作

1. **檢查 AI 狀態**
   - 開啟 http://localhost:5000
   - 查看「AI 狀態」卡片
   - 綠色表示 AI 可用，紅色表示需要設定

2. **設定 AI 模型**
   - 點擊「設定 AI」按鈕
   - 按照提示完成安裝
   - 重新整理頁面確認狀態

3. **分析郵件**
   - 點擊「分析最近郵件」
   - 等待 AI 分析完成
   - 查看分析結果和報告

## 🔧 AI 分析功能

### 1. 郵件摘要
- 自動生成郵件要點
- 評估重要性（1-10分）
- 建議回覆優先級

### 2. 智慧分類
- 工作相關（專案、會議）
- 客戶溝通（詢問、報價）
- 財務相關（發票、付款）
- 人事通知（請假、公告）
- 技術支援（IT問題）
- 行銷推廣（產品介紹）
- 個人郵件
- 其他

### 3. 關鍵字提取
- 自動識別重要關鍵字
- 支援中文關鍵字
- 最多提取10個關鍵詞

### 4. 情感分析
- 識別情感傾向（正面/中性/負面）
- 評估情感強度（1-10分）
- 提取主要情感詞彙

## ⚙️ 配置檔案

AI 配置儲存在 `ai_config.json`：

```json
{
  "ollama": {
    "url": "http://localhost:11434",
    "model": "llama3:8b",
    "timeout": 30
  },
  "analysis": {
    "max_email_length": 1000,
    "batch_size": 5,
    "cache_results": true
  }
}
```

## 🛠️ 故障排除

### 常見問題

**Q: AI 狀態顯示「未設定」**
A: 請執行 `python setup_ai.py` 完成安裝

**Q: 模型下載失敗**
A: 檢查網路連接，確保有足夠的儲存空間

**Q: 分析速度很慢**
A: 這是正常的，本地模型需要時間處理，建議批次分析

**Q: 記憶體不足**
A: 考慮使用更小的模型或升級記憶體

### 進階設定

**更換模型**
```bash
# 下載其他模型
ollama pull llama3:70b  # 更大但更準確
ollama pull qwen:7b     # 中文優化模型
```

**調整配置**
編輯 `ai_config.json` 修改參數：
- `max_email_length`: 郵件最大長度
- `timeout`: API 超時時間
- `batch_size`: 批次處理大小

## 📈 效能優化

1. **批次分析**: 一次分析多封郵件更有效率
2. **內容限制**: 限制分析內文長度提升速度
3. **結果快取**: 重複分析相同郵件時使用快取

## 🔒 隱私與安全

- 所有 AI 分析都在本地進行
- 不會將郵件內容傳送到外部服務
- 模型檔案儲存在本地
- 完全離線運作（安裝完成後）
