# ⚡ 速度優化指南

## 🐌 **為什麼速度慢？**

### **主要原因**
1. **📧 郵件數量太多**: 您有 21,320 封郵件
2. **💻 電腦處理限制**: 每封郵件都要讀取和處理
3. **🔄 重複處理**: 沒有快取機制
4. **🌐 網頁載入**: 一次顯示太多資料

---

## 🚀 **解決方案**

### **方案1: 使用快速版本（推薦）**

```bash
# 使用快速版本 - 只處理 50 封郵件
python fast_outlook_builder.py
```

**優點:**
- ⚡ 只需 0.7 秒
- 📊 足夠日常使用
- 🔄 可以定期更新

### **方案2: 減少處理數量**

修改 `outlook_db_builder.py`:
```python
MAX_EMAILS = 50  # 從 500 改為 50
```

### **方案3: 分批處理**

```bash
# 第一次處理最近 100 封
python outlook_db_builder.py

# 需要更多時再增加數量
# 修改 MAX_EMAILS = 200
python outlook_db_builder.py
```
